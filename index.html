Troubleshooting

If you're having issues with the main application, try these steps:

  1. Make sure all dependencies are installed by running install_dependencies.bat
  2. Check that your .env file has all required values
  3. Try running the simple server with python simple_server.py
  4. For Windows production mode, use run_windows_production.bat
