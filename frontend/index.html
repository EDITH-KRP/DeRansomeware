How It Works

Real-time Detection

Our system monitors file changes in real-time to detect suspicious encryption patterns typical of ransomware attacks.

Decentralized Backups

Critical files are automatically backed up to IPFS via Filebase, ensuring they remain accessible even if local copies are compromised.

Blockchain Verification

All security events are logged to the Ethereum blockchain, creating an immutable record of file activity and potential threats.
